doi2bib
=======

### Installation
```bash
clone this repository
cd doi2bib2
npm install
```

### Run Development
```bash
npm run dev
open http://localhost:3000
```

### Create Distribution
```bash
npm run build
```

### Run Distribution
```bash
npm start
open http://localhost:3001
```

### Deploy
```bash
now # new now domain is copied to clipboard
now alias <ctrl-v> doi2bib.org
now alias <ctrl-v> www.doi2bib.org
```
